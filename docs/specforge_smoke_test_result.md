## Tiny Acceptance Check

After training, we also launched SGLang with the smoke checkpoint and ran `benchmarks/bench_eagle3.py` on one GSM8K prompt.

Command shape:

```bash
cd reference/code/SpecForge
python benchmarks/bench_eagle3.py \
  --model-path Qwen/Qwen3-0.6B \
  --speculative-algorithm EAGLE3 \
  --speculative-draft-model-path outputs/rl_project/smoke-qwen3-0.6b-eagle3/epoch_0_step_5 \
  --config-list 1,0,0,0 1,3,1,4 \
  --benchmark-list gsm8k:1 \
  --dtype bfloat16 \
  --tp-size 1 \
  --trust-remote-code \
  --name smoke_acceptance \
  --output-dir results/rl_project
```
